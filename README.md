# Sales Trend Analysis Using SQL

## Project Overview

This project analyzes sales trends using the Olist Brazilian E-commerce dataset in PostgreSQL. The analysis focuses on monthly revenue, order volume, customer purchasing behavior, and sales performance using SQL queries.

---

## Objectives

- Analyze monthly sales trends.
- Calculate total monthly revenue.
- Measure monthly order volume.
- Identify top-performing months.
- Analyze customer purchasing behavior.
- Practice SQL aggregations, joins, and date functions.

---

## Queries Performed

### 1. Customer Order Analysis
Counted the number of orders placed by each customer.

### 2. Repeat Customer Analysis
Identified customers who placed more than one order.

### 3. Monthly Revenue & Order Volume
Calculated monthly revenue and total number of orders using SQL aggregations.

### 4. Top 5 Revenue Months
Identified the five highest-performing months based on revenue.

### 5. Monthly Sales Analysis (2018)
Analyzed monthly revenue and order volume for the year 2018.

### 6. Average Order Value
Calculated the average order value for each month.
